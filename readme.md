# Controle Financeiro Caio & Itallo

Sistema simples para controle de empréstimos e pagamentos entre amigos.

## Como usar

1. Acesse: https://[seu-usuario].github.io/controle-financeiro-caio-itallo/
2. Use a senha: **financa123**
3. Registre as movimentações:
   - Empréstimos (uma pessoa para outra)
   - Pagamentos (devolução do valor)
4. Acompanhe os saldos em tempo real

## Funcionalidades

- ✅ Registrar empréstimos e pagamentos
- ✅ Ver saldo atual entre Caio e Itallo
- ✅ Histórico completo de movimentações
- ✅ Proteção por senha
- ✅ Exportar/importar dados
- ✅ Funciona offline
- ✅ Responsivo (mobile e desktop)

## Acessar de outro dispositivo

1. No dispositivo atual: Exporte os dados (botão "Exportar Dados")
2. Envie o arquivo `.json` para o outro dispositivo
3. No novo dispositivo: Importe os dados (botão "Importar Dados")

## Dados

- Os dados são salvos localmente no seu navegador
- Backup automático diário
- Senha: financa123 (ambos usam a mesma)
